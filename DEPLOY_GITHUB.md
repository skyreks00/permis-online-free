# Déploiement sur GitHub Pages

1. Assure-toi d'avoir tout commité sur main.
2. Pour déployer, lance :

    npm run deploy

Le site sera accessible à l'adresse :
https://skyreks00.github.io/permis-online-free/

Si tu changes le nom du repo ou du user, adapte la clé `base` dans vite.config.github.js.
